What is a Full Stack Developer?

A Full Stack Developer is someone who builds every aspect of the web application- from the visuals and interactions, to the data transfer and processing.
They primarily work with two key components of web development.
There is frontend development where the developer builds the visuals displayed on the user's end.
There is also backend development where the data and logic stored on the server.

Navigating Around the Git Bash

Different Git Bash Commands:

    pwd - This command prints the directory (place on your hard drive that stores files and folders) that you are in
    ls - This command lists the directories and objects that are in your directory
    cd Directory Name: This command changes the directory that you are working in
    cd .. : This command changes the directory to the previous directory you were in
    cd ~:This command changes the directory to the root directory
    touch name of file to create - This command makes a new file
    mkdir name of directory to create - This command makes a new folder
    rm name of file to remove - This command deletes the named file
    rm -r name of directory to remove - This command deletes the named directory
    explorer name of file - This command opens the specified file or folder
    explorer . - This command opens all files or folders in the directory

To declare a doctype -
<!DOCTYPE>
<html>
<body>
YOUR CODE GOES IN HERE!
</body>
</html>

<head>
<title>
WHAT SHOWS IN THE BROWSER TAB GOES IN HERE!

THIS IS WHERE YOU LINK ALL YOUR EXTERNAL RESOURCES (CSS AND JS)

To create an h1 tag with a title of your choice:
<h1> YOUR TITLE GOES HERE </h1>

To embed an image -
<img src = "IMAGE LOCATION GOES HERE">

To create an ordered list (a list with bullet points) -
<li>ITEM 1 IN LIST</li>
<li>ITEM 2 IN LIST</li>
To create an unordered list (a list without bullet points) -
<ul>ITEM 1 IN LIST</ul>
<ul>ITEM 2 IN LIST</ul>
To create a table -
<table>
<tr>
<th>COLUMN NAME</th>
<th>COLUMN NAME</th>

</tr>
<tr>
<td>ITEM 1</td>
<td>ITEM 1</td>

Mind you! These tags make the text bolded and centered in a table-
<th></th>

While These tags are not bolded and centered in a table -

To separate links without line breaks -
<nobr>INSERT CODE HERE</nobr>

To embed a YouTube Music Video -

What is Git and Why use it?

Git is a distributed version control system. It allows for a person, or a group of people to work on a project, and to keep track of the versions of the project. It is an organized system for managing code. This system allows for resolving conflicts in code, and it has a version history. This allows for easy management of a project, and a way to fall back just in case something goes horribly wrong.

What is GitHub?

Git Hub is a Web Based hosting service to store code online. It allows developers to pull (download) code or push (upload) code to the same repository (directory). It also allows developers to view histories of code changes and track issues.

Basic Git Commands

    git clone : copies an entire repo (to begin)
    git add: adds a file for inclusion in Git
    git commit: notes a change to the local repo
    git push: sends changes to hosting service
    git pull: downloads freshest version of the repo
